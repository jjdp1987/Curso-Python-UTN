🚀 Tu Itinerario Final Consolidado
Como ya terminaste la configuración técnica, este es el resumen que podés guardar para siempre:

----------------------------------------------------------------------------------------
En la PC donde terminás de trabajar:
Si instalaste librerías: pip freeze > requirements.txt (Hacelo en la carpeta que corresponda: CURSO o NutriSport_Pro).

----------------------------------------------------------------------------------------
Sincronizar:
PowerShell
git add .
git commit -m "Descripción de lo que hiciste"
git push

----------------------------------------------------------------------------------------
En la PC donde empezás a trabajar:
Traer info: git pull.
----------------------------------------------------------------------------------------
Sincronizar motor: pip install -r requirements.txt.
----------------------------------------------------------------------------------------



*Nota: Si es la primera vez que abrís esa carpeta en esa PC, primero creás el venv: python -m venv venv.utn.
